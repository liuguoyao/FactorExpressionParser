在《101 Formulaic Alphas》论文中，作者已经完整列出了他们所使用的全部函数。但**实际量化研究与因子挖掘中**，业界常用的运算和函数远不止这些。下面补充那些论文未列出，但市场中极常用到的扩展，通常出现在 WorldQuant WebSim、Alphalens、各类回测框架或因子研究中。

---

### 1. 基础数学函数（更多）

| 函数                      | 作用                                     |
| ------------------------- | ---------------------------------------- |
| `exp(x)`                  | $e^x$                                    |
| `sqrt(x)`                 | 平方根                                   |
| `log1p(x)`                | $\ln(1+x)$，适用于小值防精度丢失         |
| `round(x, n)`             | 四舍五入到 n 位小数                      |
| `floor(x)` / `ceil(x)`    | 向下/向上取整                            |
| `min(x, y)` / `max(x, y)` | 两值取小/大（逐元素）                    |
| `clip(x, low, high)`      | 将 x 限制在 [low, high] 内               |
| `sigmoid(x)`              | $\frac{1}{1+e^{-x}}$，常用于机器学习因子 |

---

### 2. 时间序列函数（更丰富的窗口统计）

| 函数                                         | 作用                                 |
| -------------------------------------------- | ------------------------------------ |
| `ts_median(x, d)`                            | 过去 d 天的中位数                    |
| `ts_quantile(x, d, q)`                       | 过去 d 天的 q 分位数（如 0.25）      |
| `ts_skew(x, d)`                              | 过去 d 天的偏度                      |
| `ts_kurt(x, d)`                              | 过去 d 天的峰度                      |
| `ts_product(x, d)`                           | 过去 d 天的滚动乘积                  |
| `ts_ewma(x, halflife)`                       | 指数加权移动平均（可指定半衰期）     |
| `ts_regression(y, X, d)`                     | 用 X 对 y 做滚动回归，返回残差或系数 |
| `ts_beta(x, benchmark, d)`                   | 滚动 CAPM Beta                       |
| `ts_residual(x, benchmark, d)`               | 滚动回归残差，常用于去市场/行业影响  |
| `ts_entropy(x, d)`                           | 信息熵（WorldQuant 中有用）          |
| `ts_co_skewness(x, y, d)` / `ts_co_kurtosis` | 高阶共矩                             |

---

### 3. 横截面函数（更多截面操作）

| 函数                         | 作用                                     |
| ---------------------------- | ---------------------------------------- |
| `group_rank(x, group)`       | 按组（如行业）内排名，0~1                |
| `group_zscore(x, group)`     | 组内 z-score 标准化                      |
| `group_neutralize(x, group)` | 组内去均值（组间中性化）                 |
| `zscore(x)`                  | 横截面 z-score：`(x - mean(x)) / std(x)` |
| `demean(x)`                  | 横截面去均值                             |
| `winsorize(x, pct)`          | 缩尾处理，将极值拉回指定分位数边界       |
| `percentile(x)`              | 转换为 0~1 的分位数                      |
| `quantile_cut(x, n)`         | 将股票分成 n 个分层（返回组号）          |
| `rank_zscore(x)`             | 先将横截面排序再标准化，使分布更接近正态 |
| `scale_medmad(x)`            | 用中位数和 MAD 做稳健缩放                |

---

### 4. 逻辑、缺失值与条件

| 函数/操作                            | 作用                                   |
| ------------------------------------ | -------------------------------------- |
| `&` / `\|` / `!`                     | 逻辑与、或、非                         |
| `isnan(x)`                           | 判断是否缺失                           |
| `fillna(x, value/method)`            | 填充缺失值（前值填充 `ffill`、均值等） |
| `ffill(x)` / `bfill(x)`              | 前向/后向填充时序缺失                  |
| `if_else(cond, true_val, false_val)` | 向量化条件（部分框架用）               |
| `filter(x, mask)`                    | 按布尔掩码保留原值，否则置 NaN         |

---

### 5. 金融与行情特化

这些通常是作为**基础数据列**直接参与运算，而非独立函数，但在因子表达式中必不可少：

- `open`, `high`, `low`, `close`, `volume` 的各种变形
- `returns`（日收益率，`close / delay(close,1) - 1` 的快捷方式）
- `vwap`（成交量加权均价）
- `adv{d}`（过去 d 日均成交额，计算流动性）
- `market_cap`, `sector`, `industry`, `beta_to_index` 等基本特征
- `forward_returns{d}`（未来 d 日收益，用于定义训练目标）
