【任务】用 C++11 实现一个金融因子表达式解析器。

【因子表达式语言】
支持三类运算，所有运算定义在一个三维数组 data[时间][股票][字段] 上：

1. 逐元素运算符（element-wise，每个时间/股票独立计算）：
   + - * /  ^（幂） 比较(> < >= <= == !=)  三元( ? : )
   函数：abs(x) log(x) sign(x) signedpower(x,a) exp(x) sqrt(x)

2. 时间序列函数（沿每只股票的时间轴纵向计算）：
   delay(x, d)         -- x 滞后 d 个时间点
   delta(x, d)         -- x - delay(x,d)
   ts_sum(x, d)        -- 过去 d 天的滚动和
   ts_mean(x, d)       -- 过去 d 天的滚动均值
   ts_std(x, d)        -- 滚动标准差
   ts_corr(x, y, d)    -- 滚动相关系数
   ...（按需添加）

3. 横截面函数（在同一时间点所有股票上计算）：
   rank(x)             -- 横截面排名，输出缩放到 0~1
   scale(x, a=1)       -- 缩放使全市场 sum(|x|)=a
   zscore(x)           -- 标准化
   group_rank(x, sector) -- 行业内排名
   ind_neutralize(x, sector) -- 行业中性化

【输入】
- 字符串表达式，例如：
  "rank(ts_mean(close, 5) / delay(close, 10))"
- 变量名代表数据字段，如 close, volume, vwap 等，运行时提供对应数据。

【解析目标】
- 词法分析：将字符串拆成 Token 流（标识符、数字、运算符、括号、逗号）。
- 语法分析：生成抽象语法树（AST），能处理运算符优先级和结合性。
- 语义分析：为每个 AST 节点标注“作用域”（逐元素 / 时序 / 横截面），便于后续求值。

【优先要求】
- 使用 Pratt 解析器（TDOP）或递归下降，代码要清晰、可扩展。
- AST 节点使用继承体系（基类 Expr），并用 std::unique_ptr 管理。
- 完整的错误处理：遇到未定义函数、不匹配括号等需给出明确错误信息。
- 不依赖第三方库，仅使用 C++11 标准库。
